## 1. 公式结构对比

### WorkBuddy 总公式

```
含税单件 = MAX(
  (制造成本 + 废品 + 管理 + 包装) × (1+15%利润) × (1+13%税) / 数量,
  最低报价 ¥50
)

制造成本 = 材料费 + 加工费 + 设置费/件 + 表面处理费 + 检测费

材料费 = 单价 × 密度 × 体积(cm³)/10⁶ × (1+废料系数)
基础工时(h) = 体积/1000 × 设备基准系数
特征加时(h) = Σ(数量 × 单特征系数 × 复杂度系数)
综合工时(h) = (基础工时 + 特征加时) × (1/加工性) × 公差系数
加工费 = 综合工时 × 设备小时费率
设置费/件 = 设置时间(h) × 操作员费率 / 数量
表面处理 = 单价(¥/dm²) × 面积  或  单价(¥/kg) × 重量
```

### 本引擎总公式

```
单件 = (材料 + 调机 + 工序 + 切削 + 表面 + 装夹 − 批量折扣) + 运费 + 税

切削分钟 = 有效去除/MRR + 刀路附加 + 面数附加
工序费 = 孔/圆角/型腔 分钟 × 机时
```

### 结构相似度

| 费用块 | WorkBuddy | 本引擎 | 对齐度 |
|--------|-----------|--------|--------|
| 材料 | 体积×密度×单价×废料 | 毛坯体积×密度×单价×损耗 | ✅ 高 |
| 调机/设置 | 设置h×操作员/数量 | 调机分钟×机时/数量 | ✅ 高 |
| 加工 | 体积/1000+特征 → ×费率 | 去除/MRR+特征分钟 | ⚠️ 模型不同 |
| 表面 | ¥/dm² 或 ¥/kg | ¥/dm²×复杂度 | ✅ 高（面积） |
| 公差 | 时间×检测系数 | 未接入 | 📋 可扩展 |
| 利润/管理 | 8%+15% | 不含（成本/市场价） | 分层处理 |
| 最低报价 | ¥50 | 无 | 📋 可加 |

## 3. 设备机时对标

| 设备 | WB ¥/h | 引擎标准 | 引擎小批量 |
|------|-------:|---------:|-----------:|
| 3轴 CNC | 80 | 110 | 165 |
| 5轴 CNC | 150 | — | — |

WB ¥80/h 接近**车间成本**；引擎 ¥110 为**成本校准**；¥165 为**外协零售价**。三层定价合理，不冲突。

设置费：WB 默认 `1.5h × ¥40/h = ¥60` 摊到单件；引擎小批量调机 `12min × ¥110/h ≈ ¥22`，更贴近外协平台「工程费」。

WB 特征模型是 **加到基础工时上的系数**；引擎是 **分钟/个 × 机时**。两者可换算：

```
等效分钟 ≈ unitCoeff × complexityMult × 60  (在 baseTime=1h 基准下)
沉头孔 F15: 0.06 × 1.0 × 60 ≈ 3.6 min → 引擎 counterbored ~0.35min×scale 为工序费，另有切削
```

## 6. 采纳 / 不采纳 / 待扩展

### ✅ 建议参考并保留对齐

1. **公式分层**：材料 + 设置 + 加工 + 表面 + 税 — 与引擎一致  
2. **加工性系数表** — 校验 `machiningFactor` / MRR 比例  
3. **特征 F01~F15 分类** — 扩展 Palmetto 映射路线图  
4. **表面按 dm² 成本价** — 标准 tier 下限参考（¥5 阳极等）  
5. **设置费按数量摊薄** — 引擎已实现  

### ❌ 不宜直接照搬

1. **基础工时 = 体积/1000** — 忽略去除量，壳体件会算错（手电筒类）  
2. **材料按 L×W×H 体积** — 无近净料/壳体逻辑  
3. **利润 15% + 管理 8%** — 引擎定位成本/市场价，利润在商务层  
4. **ABS 材料 ¥8/kg** — 与当前采购价不符  

### 📋 建议下一步接入引擎

| 优先级 | 项 | 说明 |
|--------|-----|------|
| P1 | 公差等级 `T01~T05` | 下单可选 ±0.1/±0.05，乘 `timeMult` |
| P1 | 螺纹孔 F03/F04 | 识别后按表系数或 ¥3~4/孔 |
| P2 | 最低报价 ¥50 | 小批量 `minimumQuoteCny` |
| P2 | 检测费 | 高精公差时 `inspectMult × 工时 × 0.3` |
| P3 | 5轴设备 E03 | 复杂曲面自动选费率 ¥150/h |
