# 自动报价规则 v1.6.6 — 零件分型回归

```
总价 = 加工工艺费 + 加工时长费 − 批量折扣 + 运费 + 税费
```

**加工/时长以 Palmetto 特征为主**（孔、圆角、型腔、面数、去除量），不是只看包络尺寸。

行业 Excel 对标详见：[quote-benchmark-xls.md](./quote-benchmark-xls.md)  
WorkBuddy 参数化公式对标：[quote-benchmark-workbuddy.md](./quote-benchmark-workbuddy.md)  
零件分型回归：[quote-regression-typology.md](./quote-regression-typology.md)  
校验命令：`node scripts/compare-xls-benchmark.mjs` · `node scripts/compare-workbuddy-benchmark.mjs` · `node scripts/regress-part-typology.mjs`

## 零件复杂度档位

| 档位 | 判定要点 |
|---|---|
| **简单件** | 去除量小、孔≤6、面无曲面堆积、无型腔/轴；低去除板件可放宽至 15 孔 |
| **标准件** | 介于两者之间 |
| **复杂件** | 多孔(>12)、多面(>90)、多圆角、大去除量 |

## 低去除标准件（v1.6.4–1.6.5）

包络 ≤100cm³ 且去除 ≤25cm³ 且去除比 ≤25%：

| 项 | 处理 |
|---|---|
| 复杂度 | 标准（非复杂），评分阈值 0.72 |
| 开料 | 近净料；**>12孔** 时最低开料 300g |
| 装夹 | ¥0 |
| 表面 | **面积 <1.5dm²** → ¥5.5/dm²；**≥1.5dm²** → ¥48/dm² |
| 多孔薄板 | >12孔：大孔 1.2min、标准孔 0.38min（非 8min/孔） |
| 浅腔 | 型腔 2.5min（去除已在 MRR） |
| 调机 | 8min |

同系列支架：上层 1.11dm²/17孔 → 小面积价；下层 1.63dm²/8孔 → 大面积价。

## 壳体/空心件（v1.6.2）

去除量/体积 > 4 时识别为壳体：

- 开料按 **近净料**（`体积 + 去除×42%`）
- 切削按 **有效去除**（`去除×48%`）
- 塑料/壳体：免大去除附加费、装夹费 ¥0
- 塑料表面：¥11/dm²（喷砂+阳极）

## 加工时长费（特征驱动）

```
切削分钟 = (有效去除/MRR + 精加工刀路 + 多面附加) × 加工复杂度系数
```

| 系数 | 值 | 说明 |
|---|---:|---|
| 简单件 | ×0.58 | 大板少特征也便宜 |
| 标准件 | ×1.0~1.18 | 随孔/面/圆角略增 |
| 复杂件 | ×1.32~1.55 | 多孔多曲面加价 |

## 数量档位

| 数量 | 档位 | 说明 |
|---|---|---|
| **qty = 1** | 单件小批量 | 零售材料、溢价机时 ¥165/h、装夹费、平台级表面 |
| **qty ≥ 2** | 标准批量 | 成本校准机时 ¥110/h，调机摊薄 + **批量折扣** |

### 批量折扣（摘自行业报价表，v1.6.3）

仅 **标准 tier**、**qty ≥ 2**，作用于非材料费（调机+工序+切削+表面+装夹）：

| 数量 | 系数 |
|---|---:|
| 2~9 件 | ×0.90 |
| 10~30 件 | ×0.80 |
| 31 件以上 | ×0.75 |

验证成本校准：`node scripts/calibrate-quotes.mjs`（`pricingTier: standard`, qty=1）

## qty=1 小批量参数（`SMALL_BATCH_ADJUSTMENTS`）

| 参数 | 值 | 对标 |
|---|---:|---|
| 材料单价倍率 | ×1.65 | 零售采购 |
| 最低开料 | 480 g | 平台小单（简单件/壳体豁免） |
| 切削机时 | ¥165/h | 外协单件 |
| 调机机时 | ¥110/h | 工程费（~12min） |
| 装夹费 | ¥15 | 平台装夹（简单件/壳体 ¥0） |
| MRR 缩放 | ×0.55 | 非简单件；简单件 ×0.35 |
| 铝 喷砂+阳极 | ¥58/dm² | 平台表面处理 |
| 塑料 喷砂+阳极 | ¥11/dm² | 壳体塑料件 |

## qty≥2 成本参数（四单校准）

| 参数 | 值 |
|---|---:|
| 机时 | ¥110/h |
| 调机 | 26 min（标准）/ 10 min（简单件） |
| 铝 MRR | 10 |

## 表面处理（纯面积计价）

```
表面处理费 = 实际表面积(dm²) × 单价(¥/dm²) × 复杂度系数
```

无底价、无最低开单面积。

| 工艺 | 标准 ¥/dm² | 单件小批量 ¥/dm² |
|---|---:|---:|
| 喷砂+阳极 | 32 | 58 |
| 阳极 | 28 | 48 |
| 喷砂 | 18 | 32 |
| 塑料 喷砂+阳极 | — | 11 |

## 费用块

| 费用块 | 内容 |
|---|---|
| **加工工艺费** | 毛坯材料 + 调机 + 工序 + 装夹 + 表面 |
| **加工时长费** | 切削走刀（MRR×机时，不含调机） |
| **批量折扣** | 标准 tier qty≥2 时扣减 |
| **运费** | ¥4 |
| **税费** | 13% |
