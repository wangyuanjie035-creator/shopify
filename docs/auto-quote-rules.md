# 自动报价规则 v1.5 — 单件小批量 + 批量成本价

```
总价 = 加工工艺费 + 加工时长费 + 运费 + 税费
```

## 数量档位

| 数量 | 档位 | 说明 |
|---|---|---|
| **qty = 1** | 单件小批量 | 材料零售/最低开料、保守 MRR、装夹费、溢价机时、平台级表面处理 |
| **qty ≥ 2** | 标准批量 | v1.4.1 四单成本价校准，调机费随数量摊薄 |

验证成本校准：`node scripts/calibrate-quotes.mjs`（强制 `pricingTier: standard`）

## qty=1 小批量参数（`SMALL_BATCH_ADJUSTMENTS`）

| 参数 | 值 | 对标 |
|---|---:|---|
| 材料单价倍率 | ×1.65 | 零售采购 |
| 最低开料 | 480 g | 平台小单开料 |
| 切削机时 | ¥165/h | 外协单件 |
| 调机机时 | ¥110/h | 工程费（~12min≈¥20） |
| 装夹费 | ¥15 | 平台装夹 |
| MRR 缩放 | ×0.10 | 保守工时（加工费↑） |
| 阳极/喷砂阳极 | ¥76 | 平台表面处理 |

## qty≥2 成本参数（四单校准 MAPE ~1.6%）

| 参数 | 值 |
|---|---:|
| 机时 | ¥110/h |
| 调机 | 26 min（标准）/ 10 min（简单件） |
| 铝 MRR | 10 |
| 黄铜 MRR | 1.8 |

## 表面处理（v1.5.2 纯面积计价）

```
表面处理费 = 实际表面积(dm²) × 单价(¥/dm²) × 复杂度系数
```

**无底价、无最低开单面积** — 小零件表面积小则费用可低至数元。

| 工艺 | 标准 ¥/dm² | 单件小批量 ¥/dm² |
|---|---:|---:|
| 喷砂+阳极 | 32 | 58 |
| 阳极 | 28 | 48 |
| 喷砂 | 18 | 32 |

复杂度系数仍受孔/沉头/型腔/圆角及简单/复杂件档位影响。

| 费用块 | 内容 |
|---|---|
| **加工工艺费** | 毛坯材料 + 调机 + 工序 + 装夹(qty=1) + 表面 |
| **加工时长费** | 切削走刀（MRR×机时，不含调机） |
| **运费** | ¥4 |
| **税费** | 13% |

## 后台展示

- **加工时长费**：仅切削走刀
- **机时合计**：调机 + 工序 + 切削（与「预估加工时长」对应）
- **预估加工时长**：`调机X+工序Y+切削Z` 分钟

配置见 `utils/quote-engine.js`。
