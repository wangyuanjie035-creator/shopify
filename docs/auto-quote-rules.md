# 自动报价规则 v1.4.1 — 市场成本价（4 单校准）

```
成本总价 = 加工工艺费 + 加工时长费 + 运费 + 税费
```

**不含利润、无最低接单额。** 机时/MRR/特征时长以 4 单历史成本价为锚点校准（MAPE ~1.6%）。

## 校准基准（成本价）

| 零件 | 成本价 ¥ | 材料 |
|---|---:|---|
| Bracket | 124.95 | 6061 铝 |
| DoorStop | 808.43 | 6061 铝 |
| metal cup | 376.18 | H59 黄铜 |
| Carriage | 296.03 | 6061 铝 |

验证：`node scripts/calibrate-quotes.mjs`

## 市场默认参数（`MARKET_SHOP_PROFILE`，已校准）

| 参数 | 默认值 | 说明 |
|---|---:|---|
| 三轴铣机时费率 | **¥110/h** | 4 单拟合 |
| 标准调机 | **26 min** | 编程+装夹+首件 |
| 简单件调机 | **10 min** | 少特征、小去除量 |
| 毛坯损耗系数 | **1.10** | 材料费按包络毛坯 × 1.10 |
| 利润率 | **0** | 不加利润 |

## 公式

### 材料费（毛坯市场价）

```
毛坯材料费 = 包络体积(cm³) × 密度 × 1.10 × 材料单价(¥/kg) / 1000
```

### 调机费

```
调机费 = 调机分钟 ÷ 60 × 机时费率 × 材料加工系数 ÷ 数量
```

### 加工时长费（MRR 模型）

```
切削分钟 = 去除体积 ÷ MRR(材料) × 黄铜系数 + 精加工附加 + 大零件/多面附加
大去除附加 = max(0, 去除体积 - 40cm³) × ¥0.45/cm³
加工时长费 = 切削分钟 ÷ 60 × 机时费率 + 大去除附加
```

### 特征工序费

```
特征分钟 = Σ(孔/圆角/型腔/轴 个数 × 市场分钟/个) × featureTimeScale × 黄铜特征系数
特征费 = 特征分钟 ÷ 60 × 机时费率
```

## 校准 MRR（cm³/min，覆盖材料表默认值）

| 材料 | MRR | 单价 ¥/kg |
|---|---:|---:|
| 6061 铝 | **10** | 28 |
| H59 黄铜 | **1.8** | 65 |
| 45# 钢 | 3.3 | 8.5 |
| SUS304 | **4** | 25 |
| ABS 塑料 | **20** | 16 |

## 黄铜附加系数

| 参数 | 值 | 说明 |
|---|---:|---|
| brassFeatureScale | 2.5 | 孔/沉头等特征时长 |
| brassDurationFactor | 1.3 | 切削时长 |

## 四块映射

| 费用块 | 内容 |
|---|---|
| **加工工艺费** | 毛坯材料费 + 调机 + 特征工序 + 表面处理 |
| **加工时长费** | 去除体积/MRR × 机时 + 大去除附加 |
| **运费** | ¥4 |
| **税费** | 13%（成本项，非利润） |

调参脚本：`node scripts/tune-market-quotes.mjs`  
配置见 `utils/quote-engine.js`。
