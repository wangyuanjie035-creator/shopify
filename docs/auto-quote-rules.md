# 自动报价规则 v1.2

```
总价 = 加工工艺费 + 加工时长费 + 运费 + 税费
```

## 四块与主要因素映射

| 费用块 | 包含的主要因素 |
|---|---|
| **加工工艺费** | **材料费**（体积×密度×材料单价）、开机/setup、孔/圆角/轴/型腔、表面处理 |
| **加工时长费** | 去除体积、包络尺寸、面数复杂度 × **材料加工系数** |
| **运费** | 固定 ¥4 |
| **税费** | 13% |

## 材料如何影响价格

材料通过 **三条路径** 进入四块费用：

| 路径 | 计入哪一块 | 说明 |
|---|---|---|
| **材料费** | 加工工艺费 | 体积 × 密度 × 材料单价（¥/kg） |
| **加工难度系数** | 加工工艺费 + 加工时长费 | 开机、孔/圆角/型腔、去除体积 × `machiningFactor` |
| **切削特性** | 加工时长费 | 去除体积 × `removalFactor`；工序 × `featureFactor` |

| 材料大类 | 典型牌号 | 密度 g/cm³ | 单价 ¥/kg | 加工× | 去除× | 工序× |
|---|---|---:|---:|---:|---:|---:|
| 铝合金 | 6061 | 2.7 | 28 | 1.0 | 1.0 | 1.0 |
| 塑料 | ABS | 1.05 | 16 | 0.52 | 0.85 | 0.7 |
| 铜合金 | H59黄铜 | 8.5 | 65 | 1.8 | 1.0 | 1.0 |
| 合金钢 | 45#钢 | 7.85 | 8.5 | 1.1 | 1.08 | 1.0 |
| 不锈钢 | SUS304 | 7.93 | 25 | 1.15 | 1.1 | 1.0 |

同体积零件：**黄铜 > 不锈钢 > 铝合金 > 塑料**（材料费更高 + 加工更慢更贵）。

配置见 `utils/quote-engine.js` → `MATERIAL_PROFILES`。
