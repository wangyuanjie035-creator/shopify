# 自动报价规则 v1.1

价格拆为 **四块**：

```
总价 = 加工工艺费 + 加工时长费 + 运费 + 税费
税费 = (加工工艺费 + 加工时长费 + 运费) × 13%
```

## 四块含义

| 块 | 包含 |
|---|---|
| **加工工艺费** | 开机/setup、材料费、孔/圆角/轴、沉头、表面处理（**仅在选择时**） |
| **加工时长费** | 去除体积、大零件附加、面数复杂度 |
| **运费** | 固定 ¥4 |
| **税费** | 13%，单独列出 |

「**无需表面处理**」→ 表面处理费 = ¥0。

`预估加工时长` = 加工时长费 ÷ (¥150/小时)。
